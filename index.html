LibrasWidget

v1.0.0

Widget overlay de LIBRAS
para qualquer site

Adicione tradução automática para Língua Brasileira de Sinais em qualquer aplicação web com uma linha de código. Powered by VLibras.

Funcionalidades

🎯

Shadow DOM isolado

Estilos completamente isolados — não interfere com o CSS da página host.

👁️

MutationObserver

Detecta novos elementos no DOM e traduz automaticamente quando configurado.

⚡

API programática

widget.translate("texto") — chame quando quiser após qualquer evento.

Instalação

<!-- 1. Adicione antes de </body> -->
<script src="libras-widget.umd.cjs"></script>
<script>
  // 2. Inicialize com suas opções
  const widget = new LibrasWidget({
    position:     'bottom-right',   // bottom-right | bottom-left | top-right | top-left
    color:        '#a813f7',         // cor do botão
    watchSelector:'.bot-message',    // auto-traduz esses elementos quando aparecerem
    autoTranslate: true,
  });

  // 3. Ou chame manualmente (ex: ao receber resposta de uma API)
  widget.translate("Texto a ser traduzido para LIBRAS");
</script>
